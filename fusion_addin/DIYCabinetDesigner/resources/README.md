# Button-Icon

Hier kann später das Button-Icon für das Fusion-360-Add-in abgelegt werden.

Aktuell verwendet der Prototyp kein eigenes Icon. Wenn ein Icon ergänzt wird, sollte es in einem eigenen Ressourcenordner liegen und in `DIYCabinetDesigner.py` beim Aufruf von `addButtonDefinition(...)` als Ressourcenpfad übergeben werden.

Typische Fusion-360-Icon-Größen sind kleine und große PNG-Dateien für helle und dunkle Oberflächen. Die genaue Benennung sollte beim Einbau des finalen Icons gegen die aktuelle Fusion-360-Add-in-Dokumentation geprüft werden.
